# ML Foundations Bootcamp

Course materials for the Machine Learning Foundations bootcamp through [Codecademy Live Learning](https://www.codecademy.com/live). A 20-session, 10-week course covering classical machine learning through deep learning fundamentals.

## About the Course

This bootcamp takes you from zero ML experience to building and training neural networks. The first half covers classical ML (regression, classification, trees, ensembles, clustering, dimensionality reduction), and the second half covers deep learning (neural networks, CNNs, transfer learning, sequence models, transformers).

Live sessions run on Zoom with a mix of lecture, live demos, and hands-on coding. These notebooks are the primary hands-on materials.

## How to Use These Notebooks

Each notebook is designed to run in **Google Colab** with no local setup required.

1. Click the Colab link for the session you want
2. Save a copy to your own Google Drive (`File → Save a copy in Drive`)
3. Run the setup cell first, then follow along

You can also clone this repo and run notebooks locally with Jupyter, but you'll need Python 3.8+ and the libraries listed in each notebook's setup cell.

## Curriculum Overview

**Weeks 1-3: Foundations & Linear Models**
Regression, classification, logistic regression, regularization, evaluation metrics

**Weeks 4-5: Beyond Linear Models**
Decision trees, ensembles, SVMs, KNN, K-Means, PCA

**Week 6: Synthesis**
End-to-end ML workflow, classical ML review

**Weeks 7-8: Deep Learning**
Neural networks, backpropagation, CNNs, transfer learning

**Weeks 9-10: Advanced Topics & Career**
Sequence models, transformers, career prep, capstone presentations
